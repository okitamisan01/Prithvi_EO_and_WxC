## ✅ Step-by-Step Setup Order

**Step 1 — Set High Performance Power Plan**
- Press `Win + R` → type `powercfg.cpl` → Enter
- Select **High Performance** (or Ultimate Performance if available)

powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

---

**Step 2 — Disable Sleep via PowerShell**
- Open PowerShell as **Administrator**
- Run:
```powershell
powercfg /change standby-timeout-ac 0
powercfg /change monitor-timeout-ac 0
```

---

**Step 3 — Install PowerToys & Enable Awake**
- Download from: https://github.com/microsoft/PowerToys/releases
- Install → open PowerToys → go to **Awake** → toggle ON → set to **"Keep awake indefinitely"**

---

**Step 4 — Add Auto-Shutdown to your `train.py`**
- Add at the very end of your script:
```python
import os
os.system("shutdown /s /t 60")
```

---

**Step 5 — Launch Training with Logging**
- Open PowerShell in your project folder and run:
```powershell
python train.py > training_log.txt 2>&1
```

---

**Step 6 — After Training is Done, Restore Power Settings**
```powershell
powercfg /change standby-timeout-ac 30
powercfg /change monitor-timeout-ac 15
```
*(This runs automatically before shutdown if you add it before the shutdown command in `train.py`)*

---

Do Steps 1–3 **before** you start training, and Steps 4–5 are your launch. Step 6 happens automatically at the end. Let me know if you get stuck on any step!
